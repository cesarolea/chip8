# chip8-clj

![horns.ch8 running in chip8-clj](https://raw.githubusercontent.com/cesarolea/chip8-clj/master/resources/horns.png)

chip8-clj is a Chip8 interpreter written in Clojure.

## Usage

In the REPL:

```
(-main)
(read-rom-file "/path/to/a/rom")
(cpu/resume)
```
